EchoSync

AI-Assisted Pre-Arrival Intelligence Dashboard

cd ../projects

The Problem

In Singapore, over 2,500 people suffer Out-of-Hospital Cardiac Arrests (OHCA) annually. The most dangerous gap in emergency response is the "no-one-calls-for-help" blind spot during unwitnessed emergencies in residential HDBs. Existing solutions require manual activation, leading to delayed Community First Responder (CFR) deployment and devastatingly low survival rates.

The Dual-Pitch Solution

I led the UI/UX design and Frontend architecture for EchoSync: a privacy-first, software-simulated ecosystem that translates edge-sensor anomalies into explainable alerts. This allows for earlier intervention without continuous video surveillance. The frontend ecosystem was adapted for two high-impact proposals:

  • SCDF x Dell Innovation Challenge: Built a simulated dispatcher dashboard that receives high-confidence AI alerts and triggers a myResponder-aligned CFR workflow, prioritizing AED coordination before paramedics arrive.
  • NTU Pinnacle Prize: Optimized the interface for Smart City elderly care and non-invasive health monitoring, shifting focus from reactive response to preventive community protection.

MVP Frontend Architecture

  • > Dispatcher Dashboard: React/Next.js interface displaying explainable AI confidence scores to reduce operator cognitive load.
  • > CFR Routing UI: Mobile-responsive interface for Community First Responders, routing them to the exact unit and nearby AED staging points.
  • > Vitals Relay Simulation: UI components to relay secure ECG telemetry from community AEDs directly to the SCDF dashboard.
  • > Human-in-the-loop: Interface design ensures operators retain full override authority over alert escalations.

Dashboard React Snippet

// SCDF Dispatcher Alert Rendering & Confidence Verification
import React, { useState, useEffect } from 'react';
import { AlertCard, CFRDispatcher } from '@/components/emergency';

export default function IncomingAlerts() {
    const [incidents, setIncidents] = useState([]);

    useEffect(() => {
        // Listen for edge-processed anomaly alerts
        const alertStream = new WebSocket('wss://api.echosync.sg/alerts');
        
        alertStream.onmessage = (event) => {
            const data = JSON.parse(event.data);
            // Only surface High-Confidence alerts to reduce dispatcher fatigue
            if (data.confidenceScore >= 0.85 && data.type === 'CRITICAL_MEDICAL') {
                setIncidents(prev => [data, ...prev]);
            }
        };
        return () => alertStream.close();
    }, []);

    return (
        <div className="incident-feed">
            {incidents.map(incident => (
                <AlertCard key={incident.id} data={incident}>
                    <CFRDispatcher location={incident.hdbUnit} />
                </AlertCard>
            ))}
        </div>
    );
}
